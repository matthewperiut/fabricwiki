Adding a block

How to add a block, model and texture

Creating a basic block

You don't necessarily need a custom class for your block, that's only if you require extra functionality.

It's recommended you store all of your block instances in their own class; ModBlocks for example


    public class ModBlocks {
        // This will inherit all the properties of stone
        public static final Block TEST = new Block(FabricBlockSettings.copyOf(Blocks.STONE));
    }

You may have noticed you can't give yourself the block, that's because you need to create a block item. Items and their settings are covered more in the Adding an item section 


    public class ModItems {
        public static final Item TEST = new BlockItem(ModBlocks.TEST, new FabricItemSettings());
    }

Last thing to do is register them, I'm doing this in our mod initializer; it's up to you to organise this better


    public class Tutorial implements ModInitializer {
        ...
        @Override
        public void onInitialize() {
            Registry.register(Registry.BLOCK, new Identifier(MOD_ID, "test"), ModBlocks.TEST);
            Registry.register(Registry.ITEM, new Identifier(MOD_ID, "test"), ModItems.TEST);
        }
    }
